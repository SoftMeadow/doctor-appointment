# 🩺 Запись к врачу (Haskell)

Приложение на Haskell с графическим интерфейсом (Monomer), предназначенное для выбора врача на основе выбранных пользователем симптомов.

Пользователь проходит по иерархическому дереву жалоб, выбирает симптомы, после чего программа формирует рекомендацию врача с пояснением.

---

## 🚀 Возможности

- Иерархический выбор жалоб (многоуровневое дерево)
- Взвешенная система рекомендаций:
  - тревожные симптомы (red flags)
  - ключевые симптомы
  - обычные и второстепенные симптомы
- Автоматическое обновление рекомендации
- Обоснование результата (совпавшие симптомы, баллы)
- Возможность добавления и удаления симптомов
- Отображение пути по дереву жалоб
- Проверка корректности данных (дубликаты, ссылки, циклы)
- Графический интерфейс на Monomer

---
